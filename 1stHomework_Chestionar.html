1. HTTP este un protocol de comunicare (transportare de date) creat pentru a conecta browser-ele cu serverele care gazduiesc site-uri web. Raspundeti la intrebarea: - Ce reprezinta un protocol de comunicare? * Un protocol în comunicare se referă la un sistem de reguli care definesc modul în care datele sunt schimbate în interiorul sau între computere. Este esențial ca dispozitivele să convină asupra formatului datelor care sunt schimbate pentru a stabili o comunicare eficientă. Setul de reguli care definește acest format este cunoscut sub numele de protocol. Protocoalele asigură că datele sunt transmise și primite corect, permițând dispozitivelor să înțeleagă și să interpreteze informațiile partajate. Acestea oferă un cadru standardizat pentru comunicare, permițând diferitelor dispozitive și sisteme să interacționeze fără probleme. (sursa:https://developer.mozilla.org/en-US/plus/ai-help) 2. Comunicarea dintre client (browser) si server (website) seamana cu un dialog. Raspundeti in termeni tehnici in limba Engleza - Clientul trimite un "HTTP request to the server", cu alte cuinte "the URL of the resource". - Serverul trimite ca rezultat "the requested web page or resource". 3. Un website reprezinta un set de pagini legate intre ele si o aplicatie disponibila printr-o adresa unica sau o denumire de domeniu in spatiul public al internetului. - Ce reprezinta un domeniu? * Un domeniu este adresa unui site web pe internet. Este o autoritate din cadrul internetului care își controlează propriile resurse. Numele de domeniu este o modalitate de a vă adresa acestei autorități ca parte a ierarhiei într-o adresă URL. De obicei, este cea mai memorabilă parte a adresei URL, cum ar fi un nume de marcă. Un nume de domeniu complet calificat (FQDN) conține toate părțile necesare pentru a căuta această autoritate după nume fără ambiguitate, folosind sistemul de nume de domeniu al internetului. Este format din mai multe componente, inclusiv domeniul de nivel superior (TLD), domeniul de nivel al doilea (SLD) și subdomenii opționale. 4.
